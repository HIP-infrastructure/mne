## Use

Build: `docker build . -t hip-mne -f Dockerfile`
Use on a Ubuntu 20.04 host : `xhost +;docker-compose build;docker-compose up`

## Warnings

Needs to be further tested, especially regarding 3D plotting and multi-windows.
